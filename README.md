# Quan Ly CV

A production-ready Next.js dashboard scaffold with App Router, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting started

```bash
npm install
npm run dev
```

The app runs at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deployment (Vercel)

1. Push the repository to GitHub.
2. Import the repo in Vercel.
3. Ensure the root directory is set to the repository root.
4. Vercel will automatically detect the Next.js app and build with `npm run build`.

## Environment variables

Create a `.env.local` file for local development and add variables as needed:

```bash
# Example
NEXT_PUBLIC_API_BASE_URL=
```
